Mavis Beacon Teaches Eos

This is a tool to help Eos programmers learn how to touch type on an eos console, in the style of computer keyboard touch typing resources that I used as a child. Simply set it going and just follow the command line prompts!

To use this tool make sure your computer and console are on the same network and have compatible IP's and then open this showfile into Eos. Once your PC and Console are on the same LAN and have matching subnets, input the IP of the computer you are running this on into the OSC UDP TX Address box under Setup > System > Show Control > OSC, making sure OSC TX is on and the OSC UDP TX port is set to 8001.

If you are using this on the same PC as Eos, input the loopback IP address of 127.0.0.1

This is very much a work in progress, but click the button below and have fun!
